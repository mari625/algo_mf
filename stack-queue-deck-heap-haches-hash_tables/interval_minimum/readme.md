## Минимум на отрезке
ограничение по времени на тест: 1 секунда  
ограничение по памяти на тест: 256 мегабайт  

Рассмотрим последовательность целых чисел длины n. По ней с шагом 1 двигается "окно" длины k, то есть сначала в "окне" видно первые k чисел, на следующем шаге в "окне" уже будут находиться k чисел, начиная со второго, и так далее до конца последовательности. Требуется для каждого положения "окна" определить минимум в нём.

**Входные данные**  
В первой строке содержатся два целых числа n и k ($1 \leq n \leq 150000$, $1 \leq k \leq 10000$, $k \leq n$) — длины последовательности и "окна" соответственно.

На следующей строке находятся n чисел, не превосходящих 32767 по модулю, — сама последовательность.

**Выходные данные**  
Выведите n−k+1 строк — минимумы для каждого положения "окна".
