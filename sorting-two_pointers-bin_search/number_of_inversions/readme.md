## Количество инверсий
ограничение по времени на тест: 5 секунд  
ограничение по памяти на тест: 256 мегабайт  

Напишите программу, которая для заданного массива A=⟨a1,a2,…,an⟩ находит количество пар (i,j) таких, что i<j и a_i>a_j.

**Входные данные**  
Первая строка входного файла содержит натуральное число n (1≤n≤500000) — количество элементов массива. Вторая строка содержит n попарно различных элементов массива A (0≤a_i≤10^6).

**Выходные данные** 
В выходной файл выведите одно число — ответ на задачу.
